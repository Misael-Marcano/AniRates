AniRates

Plataforma para explorar anime y manga: reseñas, valoraciones compartidas, listas personalizadas, notificaciones, digest por correo y herramientas de moderación.

Ver código en GitHub README del proyecto

Sobre esta página (GitHub Pages)

Esta URL sirve una landing estática del repositorio. La aplicación Next.js usa rutas dinámicas y API en tiempo real; para la experiencia completa usa Docker Compose en local o despliega frontend + backend en un hosting con Node (por ejemplo Vercel + tu API).

Documentación en el repo

  • Documentacion.md — visión técnica y guías.
  • plan-mejoras.md — roadmap y mejoras.

Stack

  • Backend NestJS, base SQL Server, Redis opcional.
  • Frontend Next.js con i18n ES / EN / pt-BR.
  • Orquestación local con docker-compose.yml en la raíz del repo.
